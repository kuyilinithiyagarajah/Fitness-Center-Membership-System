# Fitness Center Membership System

## Project Overview
The Fitness Center Membership System is a Java desktop application developed as a group project for university coursework. The system allows users to register and renew fitness memberships while enabling administrators to manage member records efficiently.

The application is built using Java, JavaFX, and Object-Oriented Programming principles, with data handled through file-based storage.

## Key Features
- Member registration and membership renewal
- Admin management of member profiles
- Queue-based handling of renewal requests
- Insertion Sort to organize members by last renewal date
- User-friendly JavaFX interface

## Technologies Used
- Java
- JavaFX
- Object-Oriented Programming (OOP)
- Data Structures: Queue, Insertion Sort
- File I/O
- MVC architecture
